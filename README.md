# MediaPipe Hand Tracking

This is a simple real-time hand tracking demo using **MediaPipe Hands JS**.

- Live bounding box around the hand
- Runs in any modern browser (Chrome/Edge recommended)
- Works on desktop or mobile
- Can be hosted on GitHub Pages

## How to run

1. Open `index.html` in a browser, allow camera access
2. Move your hand in front of the camera
3. Bounding box will follow your hand
